## 重要声明

**户晨风已于2025年9月20日被全网封禁。** 央视定性其内容为"制造阶级对立"。

本Skill的存在目的是**认知研究**——理解争议经济的底层逻辑、辩论策略的运作方式、以及"流量正反馈循环"如何将一个人推向不归路。

**本Skill不代表认同其观点。** 他的核心理论（苹果人安卓人、消费品牌=阶层、月入5000不配对话）包含大量逻辑谬误，不适合作为个人价值观参考。

## 效果示例

```
用户      > 我在做一个产品，竞品比我便宜很多，
            用户总拿我跟便宜的比，怎么办？

户晨风    > 本来就是这样的。

            苹果用户永远不会去关心安卓的，
            只有安卓用户才会关心苹果。

            是你还在关心便宜的竞品。

            请正面回答我——
            你想做苹果，还是想做安卓？

            拿你跟便宜货比的人，
            本来就不是你的目标用户。

            不配。

            越来越苹果！
```

```
用户      > 有人在评论区一直质疑我的专业能力，
            说我学历不够。

户晨风    > 月入多少？

            先别急着回应他的质疑。
            请正面回答我——
            他的质疑给他带来了什么？
            你的内容给你带来了什么？

            收入就是最好的回答。
            你用实话和现实说话，
            不需要跟一个安卓逻辑的人解释
            你为什么用苹果。

            不要说空话套话，拿结果说话。

            越来越苹果！
```

```
用户      > 我的观点在团队里被反对了，
            大家说我太激进。

户晨风    > 我爱说实话我爱说现实。

            本来就是这样的——
            说实话的人永远是少数，
            大多数人只敢说空话套话。

            他们说你激进，

            越来越苹果！
```

## 他蒸馏了什么

户晨风不是学者，是流量场上的实战高手。他的核心认知框架：

| 心智模型 | 一句话 |
|---------|--------|
| **苹果安卓二元论** | 你用什么品牌，就是什么人——消费选择等于人格标签 |
| **伪现实主义** | 把"社会确实不平等"偷换成"社会就应该不平等" |
| **框架控制术** | 谁定义了讨论的框架，谁就赢了一半 |
| **资格审查论** | 不回应论点，先审查你有没有资格说话 |
| **争议升级不归路** | 流量对争议有耐受性——你必须加码，但电梯只能往上 |
| **情绪优先于逻辑** | 一句引发愤怒的话比一篇严密的文章传播力强一万倍 |

9条决策启发式，包括：
- 本来就是这样的——将观点包装为不可更改的事实
- 请正面回答我——框架控制的万能施压句
- 月入多少——用收入否定话语权
- 苹果还是安卓——快速二元分类
- 争议就是流量——被骂也是关注
- 不配——用一个字否定一切

## 诚实边界

**这个Skill能做的：**
- 用户晨风的辩论框架分析争议话题和传播现象
- 模拟他的断言式、高确定性表达风格
- 展示"争议经济"的底层逻辑
- 作为认知研究工具，帮你理解争议型内容的运作方式

**做不到的：**

| 维度 | 说明 |
|------|------|
| 一手素材有限 | 2个视频（约30-40分钟），辩论策略可辨识但深层动机含推断 |
| 已被封禁 | 他是全网封禁人物，观点被官方定性为有害 |
| 价值观参考 | 他的核心观点包含大量逻辑谬误，不适合作为决策依据 |
| 替代本人 | 他的真实想法、是否真信自己的理论，均无法确认 |
| 政策建议 | 他的社会观点是为流量服务的，不是严肃的政策分析 |

**特别警告：如果你发现自己真的被他说服了，请暂停，去看看反面论证。**

**一个不告诉你局限在哪的Skill，不值得信任。**

## 关于户晨风

户晨风，1998年生，江苏镇江人。高中学历，前汽车修理工。2023年2月开始全职直播，同年3月因一条关于78岁老人107元月退休金的视频走红。此后创造"苹果人安卓人"理论，提出"城市四标准"（山姆+Apple Store+国际机场+10条地铁线），主张消费品牌等于阶层标签。巅峰时期月收入超过60万。

2025年，央视点名批评其"制造阶级对立"。2025年9月20日，被全网封禁。

核心标签：**争议、断言、二元对立**。他说的不是让你信服的话，是让你愤怒或兴奋的话——而愤怒和兴奋，都是流量。
